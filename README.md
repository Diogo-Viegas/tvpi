# TVPi 📺

Aplicação estilo TV Time feita em Flask para Raspberry Pi Zero 2W.

## Funcionalidades

- Adicionar séries
- Editar / apagar séries
- Adicionar episódios
- Marcar episódios como vistos
- Progresso automático

## Stack

- Python (Flask)
- SQLite
- HTML/CSS

## Como correr

```bash
python3 -m venv venv
source venv/bin/activate
pip install flask
python3 app.py
